1. Introdução

Bem-vindo ao App de Finanças Domésticas! Esta ferramenta ajuda famílias a gerenciarem finanças compartilhadas com transparência. Seja dividindo o aluguel em 50/50 ou usando divisões complexas, nós cuidamos da matemática para que você possa focar em seus objetivos.

Navegue rapidamente entre todas as funções principais usando o menu lateral.

3. Gerenciar Membros

Para adicionar um novo membro, toque no botão + na tela da Casa. Os membros são a base de todos os cálculos.

Adicionando um novo membro com nome e cor personalizados.

Membros Regulares: Normalmente os principais ganhadores, cujas rendas são usadas para calcular os ratios de divisão.
Membros Especiais: Usados para dependentes ou fundos compartilhados (como um 'Fundo Comum'). Suas rendas (pensões, auxílios) são deduzidas das despesas totais antes do cálculo da divisão.

6. Gerenciar Metas e Orçamentos

A seção de Metas é uma ferramenta poderosa tanto para economizar para seus sonhos quanto para limitar gastos por meio de orçamentos.

Tela principal de metas mostrando barras de progresso e transações recentes.

Criar uma Meta de Economia

Ao criar uma meta de economia, você pode definir um valor alvo e até incluir juros esperados. Isso ajuda você a se manter motivado enquanto vê a barra de progresso encher.

Configurando uma nova meta de economia com um alvo específico.

Criar um Orçamento

Orçamentos são metas "invertidas". Em vez de economizar para um alvo, você define um limite para uma categoria ou período específico. O app avisa quando você se aproxima do seu limite.

Definindo um limite máximo de gastos para um orçamento.

Gerenciar o Progresso

Adicione transações manualmente ou configure Transações Recorrentes para contribuições regulares de economia ou reinícios mensais de orçamento.

Automatize suas finanças com transações recorrentes de metas.
Registre pagamentos de juros para ver suas economias crescerem mais rápido.

Marcos e Tendências

Celebre seu progresso! O app fornece feedback quando você atinge marcos como estar "Na metade" de um orçamento ou "Quase lá" com suas economias.

Use a visualização de Tendências para ver datas estimadas de conclusão para economias ou dias restantes para um orçamento com base no seu histórico de transações.

7. Modelos de Contribuição

Escolha a estratégia que melhor se adapta aos valores da sua casa. Você pode alterar isso a qualquer momento nas configurações.

ModeloDescrição
Divisão 50/50Parte igual para todos, independentemente da renda.
Divisão RelativaProporcional à parte de cada membro na renda total.
Valor FixoCada membro fica com uma quantia fixa; o restante vai para o fundo comum.
Conta ConjuntaToda a renda vai para um único fundo; todas as despesas saem de lá.
Alterne entre estratégias de contribuição com um único toque.

8. Configurações e Segurança

Sua privacidade é primordial. Todos os dados são criptografados usando SQLCipher e armazenados apenas no seu dispositivo. Nós nunca vemos seus dados financeiros.

  • Biometria: Ative a impressão digital ou reconhecimento facial para segurança extra.
  • Gerenciamento de Dados: Exporte para CSV para análises externas ou crie backups criptografados.
  • Localização: Personalize o símbolo da sua moeda e o idioma do app.

9. Melhores Práticas

Dica Pro: Use o "Membro Compartilhado" para despesas que beneficiam toda a casa, como compras ou contas de serviços públicos.
  • Frequência importa: Certifique-se de que todas as rendas e despesas estejam na mesma frequência (ex: Mensal) para uma comparação mais precisa na "Visão Geral da Casa".
  • Revise as Tendências: Verifique a seção de Tendências uma vez por mês para ver se seus hábitos de consumo estão mudando.
  • Backup frequente: Como os dados são locais, perder o dispositivo significa perder os dados. Use a função de Backup Criptografado regularmente!
