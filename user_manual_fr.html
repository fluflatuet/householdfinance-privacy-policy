1. Introduction

Bienvenue sur l'application de Finances Familiales ! Cet outil aide les ménages à gérer leurs finances partagées en toute transparence. Qu'il s'agisse d'une répartition équitable à 50/50 ou de calculs proportionnels plus complexes, nous nous occupons des calculs pour que vous puissiez vous concentrer sur vos objectifs.

Naviguez rapidement entre toutes les fonctionnalités majeures grâce au menu latéral.

2. Prise en main

La Vue d'ensemble du foyer est votre tableau de bord central pour obtenir une vue globale. Elle affiche un résumé des revenus, des dépenses et, plus important encore, la contribution calculée pour chaque membre selon la stratégie choisie.

L'écran Vue d'ensemble du foyer affiche un résumé financier consolidé.

3. Gérer les membres

Pour ajouter un nouveau membre, appuyez sur le bouton + sur l'écran du foyer. Les membres constituent la base de tous les calculs.

Ajout d'un nouveau membre avec un nom et une couleur personnalisés.

Membres réguliers : Généralement les principaux salariés dont les revenus servent à calculer les ratios de répartition.
Membres spéciaux : Utilisés pour les personnes à charge ou les fonds partagés (comme une « Cagnotte commune »). Leurs revenus (argent de poche, allocations) sont déduits du total des dépenses avant le calcul de la répartition.

6. Gérer les objectifs et les budgets

La section Objectifs est un outil puissant, idéal pour épargner en vue de réaliser vos projets ou pour limiter vos dépenses grâce à des budgets.

Écran principal des objectifs affichant les barres de progression et les transactions récentes.

Créer un objectif d'épargne

Lors de la création d'un objectif d'épargne, vous pouvez définir un montant cible et même inclure les intérêts attendus. Cela vous aide à rester motivé à mesure que la barre de progression se remplit.

Configuration d'un nouvel objectif d'épargne avec une cible spécifique.

Créer un budget

Les budgets sont des objectifs « inversés ». Au lieu d'épargner pour atteindre une cible, vous fixez une limite pour une catégorie ou une période donnée. L'application vous avertit lorsque vous approchez de votre limite.

Définition d'une limite de dépenses maximale pour un budget.

Gérer la progression

Ajoutez des transactions manuellement ou configurez des Transactions récurrentes pour des versements d'épargne réguliers ou des réinitialisations budgétaires mensuelles.

Automatisez vos finances avec des transactions d'objectif récurrentes.
Enregistrez les paiements d'intérêts pour voir votre épargne fructifier plus rapidement.

Étapes clés et tendances

Célébrez vos progrès ! L'application vous informe lorsque vous atteignez des étapes clés, par exemple lorsque vous êtes à la « Moitié du chemin » d'un budget ou « Presque au but » pour votre épargne.

Utilisez la vue Tendances pour consulter les dates d'achèvement prévues de votre épargne ou le nombre de jours restants pour un budget, sur la base de l'historique de vos transactions.

8. Paramètres et sécurité

Votre confidentialité est notre priorité absolue. Toutes les données sont chiffrées à l'aide de SQLCipher et stockées uniquement sur votre appareil. Nous n'avons jamais accès à vos données financières.

  • Biométrie : Activez le déverrouillage par empreinte digitale ou reconnaissance faciale pour une sécurité accrue.
  • Gestion des données : Exportez au format CSV pour des analyses externes ou créez des sauvegardes chiffrées.
  • Localisation : Personnalisez le symbole de votre devise et la langue de l'application.

9. Bonnes pratiques

Astuce de pro : Utilisez le « Membre Partagé » pour les dépenses qui profitent à l'ensemble du foyer, comme les courses ou les factures d'énergie.
  • La fréquence est essentielle : Assurez-vous que tous les revenus et dépenses sont configurés sur la même fréquence (ex: Mensuelle) pour obtenir une comparaison des plus précises dans la « Vue d'ensemble du foyer ».
  • Consultez les tendances : Vérifiez la section Tendances une fois par mois pour voir si vos habitudes de consommation évoluent.
  • Sauvegardez régulièrement : Les données étant locales, la perte de votre appareil entraîne la perte de vos données. Utilisez régulièrement la fonction Sauvegarde Chiffrée !

10. Guide technique : Formats d'importation CSV

L'application vous permet d'importer des données au format CSV (valeurs séparées par des virgules). Tous les imports nécessitent une ligne d'en-tête exacte en première ligne. Vous trouverez ci-dessous les colonnes requises, les types de données et des exemples pour chaque type d'importation.

Types de données courants

  • Amount (Montant) : Nombres décimaux. Utilisez un point (.) ou une virgule (,) comme séparateur décimal. Les séparateurs de milliers sont automatiquement supprimés. Exemples : 1000.50, 1000,50, 1,000.50.
  • Date (Date) : Doit être au format ISO 8601 (AAAA-MM-JJ). Si le champ est vide, la date du jour est utilisée. Exemple : 2023-10-24.
  • FrequencyFactor (Facteur de fréquence) : Un multiplicateur représentant la fréquence d'une entrée par mois. Ponctuel = 0.0, Hebdomadaire = 0.25, Mensuel = 1.0, Annuel = 12.
  • Color (Couleur) : Un entier Long ARGB 32 bits représentant la couleur hexadécimale. Exemple : 4284612846 (ce qui correspond à 0xFF6200EE).
  • Boolean (Booléen) : true ou false.

Members (Membres)

En-tête : Name, Type, Color

Règles : Type 1 = Régulier, Type 2 = Spécial.
